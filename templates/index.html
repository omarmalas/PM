Welcome to the Password Manager

Securely manage your passwords with our Password Manager web application. This application provides features for user registration, login, password generation, password strength checking, user dashboards, and a hash cracker tool.

Features:

  • Register - Create an account to start managing your passwords.
  • Login - Log in to your account securely.
  • Generate Password - Create strong and secure passwords.
  • Check Password Strength - Evaluate the strength of your passwords.
  • User Dashboard - Manage your saved passwords.
  • Logout - Log out of your account.
  • Hash Cracker - Identify and crack password hashes.
